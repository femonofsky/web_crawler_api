#  Web Crawler API
This application that provide an API for the consumer to search company information.
The consumer can filter the results by any fields. 
 
## Requirements
* Python 3.4+
* Postgres 9.4.20+
* Works on Linux, Windows, Mac OSX, BSD


### Installing

Install dependence 
```
pip install -r requirements.txt
```

## Running the tests
```
python -m pytest
```
